# To Do List

This project is a simple but dynamic to do list. 

![todolist](https://user-images.githubusercontent.com/98186427/176753615-228c6472-82b8-4b78-86e1-dcd1382bf4d0.png)

## 📝 Features
* Add and delete task
* Mark and unmark task as done
* Local storage

## 💻 Technologies
* HTML
* CSS (BEM methodology)
* JavaScript
